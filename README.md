# nvim.sh

neovim plugin directory search from the terminal

## Usage

Help response

```bash
curl https://nvim.sh
```

List all plugins

```bash
curl https://nvim.sh/s
```

Search for a plugin

```bash
curl https://nvim.sh/s/statusline
```

List all tags

```bash
curl https://nvim.sh/t
```

Search for plugins based on tag

```bash
curl https://nvim.sh/t/note-taking
```
